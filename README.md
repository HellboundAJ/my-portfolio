# Portfolio Website

This is my personal portfolio website created using HTML and CSS for my UID Project Phase - https://hellboundaj.github.io/my-portfolio/.

## Pages Included

* Home
* About Me
* Skills
* Projects
* Experience
* Contact

## Features

* Clean and modern cherry blossom themed UI
* Navigation menu with functional links to all sections
* Project showcase with descriptions and links
* Skills displayed using card layout and tags
* Animated background with floating petals (CSS animations)
* Hover effects and smooth transitions
* Responsive design using CSS media queries
* Footer with useful links
